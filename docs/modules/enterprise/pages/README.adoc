Introduction
------------

image::formkiq-enterprise-logo.png[FormKiQ Enterprise,600,600]

FormKiQ Enterprise are enterprise extensions added to FormKiQ Core to bring enterprise features and functionality that are required for today's enterprises.

**FormKiQ Enterprise** modules include:

✅ link:#tag-schema-module[Document Tag Schemas Module]

✅ link:#tag-schema-custom-domains[Custom Domains Module]

✅ link:#tag-schema-antivirus[Antivirus Protection Module]

✅ Document Email Processing Module

✅ link:#tag-schema-document-ocr[Document Ocr Module]

✅ Fulltext Document Search Module

✅ Network Share and Sync Module

✅ Intelligent Document Processing Module

✅ link:#full-encryption-module[Full Encryption Module]

✅ Document Archive and Lifecycle Module

✅ link:#full-encryption-module[Single Sign-On and Custom JWT Authorizer Module]

To learn more about FormKiQ Core, please visit https://github.com/formkiq/formkiq-core. 

To learn more about FormKiQ Enterprise, please visit https://formkiq.com.
